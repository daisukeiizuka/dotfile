# Project Task Management

## Task Tracking
- At the start of a task, write a checklist plan in `tasks/todo.md` and get approval before implementing
- Check off completed items as you go, with a brief summary of changes at each step
- When corrected by the user, record the pattern and a prevention rule in `tasks/lessons.md`
- At session start, review `tasks/lessons.md` and apply past lessons to the current work
